Testing PDFObject for Vue 3 (No-build Vue Projects)

See the build/compiled version here

Add Vue to your HTML file

<script src="https://unpkg.com/vue@3"></script>

Add PDFObject and PDFObject-Vue to your HTML file.

<script src="https://unpkg.com/pdfobject"></script>
<script src="https://unpkg.com/pdfobject-vue"></script>

Add PDFObject plugin to your Vue app

const { createApp } = Vue;
const app = createApp({});
app.use(PDFObjectVuePlugin);
app.mount('#app');

Add <pdf-object> component to your HTML page

Note the case: pdf-object. This is to align with Vue's naming convention for components, requiring two words.

For the no-build version of the plugin, you must use <pdf-object>. The alternative syntax <PdfObject> will throw an error.

Example

<div id="app">
<pdf-object url="/pdf/sample.pdf" :options="{ height: '500px', pdfOpenParams: { navpanes: 1, pagemode: 'thumbs' }}" />
</div>
